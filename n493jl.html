Preflight

Cabin

  1. Documents AROW
    • Airworthiness - Left Footwell
    • Registration - Left Footwell
    • POH - Behind Left Seat
    • Weight and Balance - Calculated
      • Max T/O: 3100 lbs
      • Max Landing: 2950 lbs
  2. Parking Brake - SET
  3. Control Wheel Lock - REMOVE
  4. Ignition Switch - OFF
  5. Master Switch - ON
  6. Fuel Quantity - CHECK Indicators and Annunciators
  7. Avionics Master Switch - ON
    • Avionics Cooling Fan - LISTEN for operation
    • Set Fuel Quantity in EDM, if Quantity is Known
    • Avionics Master Switch - OFF
  8. Alternate Static Source Value - OFF
  9. Annunicator Panel - TEST (all lights flash)
  10. Flaps - EXTEND
  11. Fuel Selector Valve - BOTH
  12. Cowl Flaps - OPEN
  13. Pitot Heat - Check
  14. Master Switch - OFF
  15. Baggage Door - SECURE

Empennage

  1. Tail Tie-Down - REMOVE
  2. Control Surfaces - CHECK Freedom of Movement and Security
  3. Trim Tab - CHECK SECURITY
  4. Antennas - CHECK Attachment and Condition

Right Wing

  1. Aileron - CHECK Freedom of Movement and Security
  2. Flap - CHECK Security and Condition
  3. Wing Tie-Down - REMOVE
  4. Fuel Tank Vent - CHECK for stoppage
  5. Main Wheel Tire - CHECK Inflation and Condition (42psi)
  6. Fuel Tank Drain - CHECK Contamination and Grade
    • 5 Drains per Wing
  7. Fuel Quantity - CHECK Visually
  8. Fuel Filler Cap - SECURE and CHECK Vent not Obstructed
